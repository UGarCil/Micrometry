Micrometry

Manual

About

Micrometry is a Python-based application that assist in the process of gathering data from a live camera feed, with emphasis on the analysis of steroscopic and microscopic images.

The software is divided in four submodules, aimed at different applications:

  • Lines: For measurements using lines only
  • Curves: For measurements using Bezier curves
  • Datasets: For gathering images for classification models in machine learning implementations
  • YOLO Datasets: For gathering images and boxed labels for classification models using YOLO

To install Micrometry, clone the GitHub repository, where you will find additional details on how to create an environment to run the program.
