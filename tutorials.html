Lines

Enter the folder .../Micrometry/measure_lines/ directly from the Command Prompt

type conda activate micrometry-env

type python main.py along with any of the desired flags:

  • --ci: int
    An integer with default value 0 representing the default webcam. Change to 1, 2, ..., to match the device that corresponds to the camera connected to the microscope
  • --res: float
    Decimal point value with a default value of 0.8, representing the dimensions of the camera relative to its original size. The value of 0.8 is used instead of 1.0 because pygame increases the resolution of an input screen.

    • black
    • random
    • reflect

Datasets YOLO

Enter the folder .../Micrometry/Datasets_YOLO/ directly from the Command Prompt

type conda activate micrometry-env

type python main.py along with any of the desired flags:

  • --ci: int
    An integer with default value 0 representing the default webcam. Change to 1, 2, ..., to match the device that corresponds to the camera connected to the microscope
  • --res: float
    Decimal point value with a default value of 0.8, representing the dimensions of the camera relative to its original size. The value of 0.8 is used instead of 1.0 because pygame increases the resolution of an input screen.
  • --kc: int
    A binary value where
    • 0: erases the information on the custom classes added by user in previous runs of the program (use when starting annotations for a new project, from scratch)
    • 1: (Default) keeps the information on the custom classes added by the user in previous runs of the program (use it when adding annotations to the same dataset)
